HTML5 Overview

HTML stands forHyper text mark up language. HTML is the code that is used to structure a web page and its content.

We will see the 3 Layers of Webpage below

  1. Structure Layer
  2. Presentation Layer
  3. Behaviour Layer

Structure Layer:

It represents the content of web Page and HTML is responsible for this layer

Presentation Layer:

It decides the look and feel of web page and CSS is responsible for this layer

Behaviour Layer

It decides the reaction of user interaction and Javascript is responsible for this layer
